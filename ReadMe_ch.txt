# stm32f103c8t6_oled
基于stm32f103c8t6最小系统开发板
感谢嘉立创免费打板转接板
感谢实验室白嫖3D打印外壳

实现功能：
typeC口串口写入128×64像素图片，永久保存，可使用串口助手之类的去通信，但是我已经写了配套的exe程序，一键还挺方便的

有1.3寸oled显示屏和0.96寸oled显示屏两种版本，共用一份代码，使用时注意修改宏定义oled.h头文件中的OLED_SIZE宏定义

两者的区别可能是由于驱动芯片不一致，0.96寸的驱动芯片为ssd1306，1.3寸并不为这个，硬套用右侧会有白边，所以起始地址需要＋2

组装方法：
购买不焊接排针的stm32f103c8t6最小系统开发板，将排针朝上焊接
在PCB板上焊接oled屏幕，PCB转接板在有丝印的引脚焊接排母（尽量多，稳固）
通过排针排母实现整体硬件连接
再将3d打印外壳套上去就ok了
